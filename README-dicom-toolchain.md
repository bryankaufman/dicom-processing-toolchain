# DICOM Processing Toolchain

Open-source DICOM utilities wrapper including DCMTK, dcm4che, and pydicom integration for medical imaging workflows.

## Features

- **DCMTK Integration**: C++ DICOM toolkit
- **dcm4che**: Java DICOM library
- **pydicom**: Python DICOM processing
- **Unified CLI**: Common interface for all tools

## Installation

```bash
pip install -r requirements.txt
```

## Usage

```bash
# Example usage
python dicom_tool.py --help
```
